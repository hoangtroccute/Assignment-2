CS303 - Assignment 2
--------------------

Q1: Single Linked List
----------------------
Implements push_front, push_back, pop_front, pop_back, insert, remove, find, and print_list functions.

Q2: Stack Using Vector
----------------------
Implements push, pop, top, print, and average functions.

--------------------
How to Run (Visual Studio)
--------------------
1. Open Visual Studio ? Create a new C++ Console Application project.
2. Add all .h and .cpp files to the project.
3. Make sure "main.cpp" is set as the startup file.
4. Press Ctrl + F5 (Run without Debugging).
